    Projects

    ClassSync

    • Working with four other students, we developed a UCLA Degree Audit Report (DAR) website tailored for CS students. The traditional DAR system is notoriously slow and difficult to use. Our website aims to provide a more modern interface and improve navigation responsiveness.
    • My contribution: I established the integration between Firebase and React, enabling seamless frontend and backend communication for our project. I designed and implemented fundamental database operations for my team, ensuring smooth data handling. Additionally, I developed the website's authentication system, featuring a robust sign-up and sign-in mechanism and securely allocates a personalized set of classes to each user. I was also in charge of the backend infrastructure to ensure reliability and performance.
    • Tools used: React, HTML, CSS, Javascript, Firebase, NodeJS, Git.

    Iron Man Gauntlet

    • Designed a gauntlet within 48 hours with functional sensors and real-time data transmission to a website.
    • My contribution: Developed an Arduino-based circuit with ultrasonic, air quality, and temperature/humidity sensors, combined with an OLED display for real-time data, and an LED for air quality or fire warnings. Collaborated with website and CAD team members to successfully transfer data to the website in near-instant, with a delay of only 3ms. Presented the design to a panel of judges and received recognition for the Most Innovative Electrical Design.
    • Check out my team's pictures and field run of the gauntlet via the "Showcase" button. The field run involves a gauntlet wearer to go through different types of hazards/rooms (not actually dangerous for obvious reason). Me and another friend will be inside the Makerspace workshop guessing what type of room it is based on the data the sensor sends to the website which we monitor inside the Makespace workshop.
    • Tools used: Arduino Nano, Arduino IDE, Javascript

    Project Three

    • Collaborated with 4 students, we made a UCLA Degree Audit Report website that is tailored for CS students. The traditional DAR program is known to be very slow and hard to navigate. This website aims to make the interface more modern and the navigation more pleasant.
    • My contribution: I set up the connection between Firebase and React, set up basic database operations for my team members, and set up the authorization portion of the project which includes a robust sign-up/sign-in system, ensuring that each user is entitled to their own set of classes and maintaining the backend.
    • Tools used: React, HTML, CSS, Javascript, Firebase, NodeJS, Git.
